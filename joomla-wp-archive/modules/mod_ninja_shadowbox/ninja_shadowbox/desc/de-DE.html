Beschreibung:

Ninja Shadowbox macht Michael J.I. Jackson's Shadowbox für Joomla verfügbar.

Die Shadowbox ist eine cross-browser, platformübergreifende, sauber kodierte und vollständig dokumentierte Multimedia Applikation, komplett in JavaScript umgesetzt. Durch die Verwendung der Shadowbox können Webmaster eine breite Palette von Medien in allen gängingen Browsern anzeigen lassen, ohne den Besucher von der verlinkenden Seite fortschicken zu müssen.

Ninja Shadowbox erleichtert Ihnen die Arbeit mir der Shadowbox deart, dass Sie das Verhalten der Shadowbox über die erweiterten Optionen einfach, schnell und benutzerfreundlich im Adminbereich Ihrer Webseite beinflussen können.

Ninja Shadowbox stellt Ihnen die Shadowbox in vollem Umfang zur Verfügung und gibt Ihnen die Flexibilität, hinsichtlich der von Ihnen anderweitig verwendeten Joomla Erweiterungen und deren JavaScript Frameworks anzupassen.

Bedienungsanleitung

Basis Setup

  • Installieren Sie das Modul
  • Wählen Sie einen CSS Skin
  • Wählen Sie sich Ihren JavaScript Framework
  • Wählen eine von 32 Sprachen
  • Veröffentlichen Sie das Modul

Tipps und fortgeschrittene Anwendung FAQ

Ich möchte auch die anderen Optionen der Shadowbox verwenden, wie mache ich das?
Wir haben alle nützlichen Optionen, welche die Shadowbox zu bieten hat, implementiert. Aktivieren Sie einfach die „Erweiterten Parameter” in der Modulverwaltung und wählen Sie Ihre bevorzugten Einstellungen aus.
Ich verstehe das nicht. Ich hab's veröffentlicht, aber mit meinen Bildern passiert nichts.
Sie müssen, sofern noch nicht durch eine andere Erweiterung geschehen, den Shadowbox Code laden. Dies können Sie in der Modulverwaltung im Adminbereich einstellen. Fü weitere Informationen besuchen Sie bitte die Webseite des Entwicklers (Usage Instructions)
Ich möchte meinen eigenen Skin verwenden, wie mache ich das?
Hierfür gibt es den Paramester „custom” im Dropdownmenü der Skin-Einstellung. Speichern Sie Ihren eigenen Skin einfach in einem Verzeichnis, welches custom heißen muss, und laden Sie es dann auf Ihren Webspace ins folgende Verzeichnis hoch: ~/modules/mod_ninja_shadowbox/js/skin

Fehlersuche FAQ

Meine Videos werden nicht abgespielt.
Um Videos abspielen zu können, müssen Sie zunächst den passenden Player laden. Hierzu stellen Sie je nach Medium in der Modulverwaltung die Auswahl bei swf, flv, qt oder wmp auf an. Nähere Informationen erhalten Sie, wenn Sie mit der Maus über die Player-Namen fahren. Im Falle eines Flash Videos (FLV) muss zusätzlich auch die Datei flvplayer.swf geladen sein. Dies können Sie in den „Erweiterten Parametern” einstellen, welche dann auch „aktiviert” sein müssen.
Was wenn meine Seite ein völlig anderes Design hat und zu keinem der mitgelieferten Skins passt?
Wir haben Ihnen die Option bereitgestellt, ein eingenen CSS Skin zu verwenden. Laden Sie ihn einfach via (S)FTP in den Ordner ~/modules/mod_ninja_shadowbox/js/skin hoch. Ihr Skin muss jedoch in einem Verzeichnis mit dem Namen custom gespeichert sein, da Sie ihn ansonsten licht laden können.
Es treten JavaScript Fehler auf.
JavaScript Fehler werden für gewöhnlich durch Inkompatibilitäten zwischen verschiedenen Framewors ausgelöst. Wählen Sie in diesem Falle einfach einen anderen JavaScript Framework auf und versuchen Sie es erneut.
